# M. Bala Vighnesh Reddy Portfolio

A static professional portfolio built from the resume content in `assets/Bala-Vighnesh-Resume.pdf`.

## Open locally

Open `index.html` in a browser. No build step is required.

## Edit

- `index.html` contains portfolio copy and project links.
- `styles.css` controls layout, responsive behavior, and visual design.
- `script.js` renders the interactive model-field hero background.
